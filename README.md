# URBAN FLEX 

A application for efficient management of transportation operations, connecting clients, operations managers, and drivers.

## System Overview

This application streamlines transportation operations through:

- **Client Management**
  - Book and track transportation services
  - View real-time status updates
  - Manage booking history
  - Rate and review services

- **Operations Manager Dashboard**
  - Monitor fleet operations in real-time
  - Assign and dispatch drivers
  - Track vehicle locations
  - Generate performance reports
  - Manage client requests

- **Driver Interface**
  - Accept/decline assignments
  - Update delivery/pickup status
  - Navigate to destinations
  - Report incidents
  - Track working hours

## Tech Stack

- Backend: Laravel 10.x
- Frontend: TypeScript & JavaScript
- Database: MySQL
- Authentication: Laravel Sanctum
- Real-time Updates: Laravel WebSockets
